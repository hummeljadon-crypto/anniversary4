# Four-Month Heart Site

Minimal mobile-first anniversary site.

## Personalize

Open `script.js` and change:

```js
const CONFIG = {
  name: "baby girl"
};
```

Open `styles.css` and change the `J` inside the boyfriend hero if needed.

## Upload to GitHub Pages

Upload these files to the root of your repo:

- index.html
- styles.css
- script.js
- .nojekyll

Then enable GitHub Pages from your main branch.
